# Guardian

Guardian is a robust security platform designed to protect sensitive data with advanced encryption and access controls.  
It combines dashboards, forms, messaging and OTP features to ensure safe communication and workflow.  
With integrated database security, Guardian delivers end‑to‑end protection for modern applications.

---

##  Features
- **Secure Dashboards**: Monitor system activity and user interactions in real time.
- **Forms & Validation**: Collect and validate data with built‑in security checks.
- **Messaging & OTPs**: Send secure messages and one‑time passwords for authentication.
- **Database Protection**: Encrypt and safeguard sensitive information at rest and in transit.
- **Access Control**: Highly customizable authentication and authorization powered by Spring Security.

---

##  Tech Stack
- **Spring Boot** (Web, Security, Data JPA)
- **PostgreSQL** (Database)
- **Thymeleaf** (Template engine)
- **Lombok** (Reduce boilerplate code)
- **Hibernate Validator** (Bean validation)
- **JavaMailSender** (Email/OTP delivery)
